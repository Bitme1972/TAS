# TAS Customer Quick Start

1. Verify the release SHA-256 before installing or extracting it.
2. Read the EULA and release notes.
3. Install or extract TAS in the approved customer location.
4. Open Product Activation and create an activation request.
5. Send the request through the supplied vendor or offline process.
6. Import the returned `.taslic` file.
7. Load sanitised auditpol evidence or use local collection where the desktop edition permits it.
8. Generate assurance results before exporting reports.
9. Store exported evidence packs according to the customer’s retention rules.
10. Use the support-bundle instructions when requesting assistance; do not include raw evidence unless explicitly agreed.
