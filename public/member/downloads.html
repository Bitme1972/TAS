ACTIVATION FLOW

After downloading

Verify the MSI hash with certutil -hashfile TAS_Professional_v10_0_6_x64.msi SHA512.
Install and open Telemetry Assurance Studio from the Start Menu or desktop shortcut.
Use Product Activation to create tas_activation_request_*.json.
Send the request to the vendor and receive the signed .taslic plus hash sidecar.
Import the licence using Activate product. Do not manually copy it into Program Files.
