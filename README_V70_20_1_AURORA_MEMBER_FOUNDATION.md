# TAS v70.20.1 AURORA Member Foundation

This release deliberately returns to the v70.20 product direction.

## Product rule

The protected AURORA command centre remains the default application at `/`. New commercial capability is added around it rather than replacing it with a generic public website.

## Added safely

- AURORA-aligned member and edition hub at `/member/`
- paid-member download-vault layout
- Community, Professional and Consultant guidance
- licensing lifecycle and offline activation documentation
- customer quick start and SHA-256 verification guide
- contextual links inside the existing Standalone Desktop and Product Activation tabs
- honest locked states for commercial binaries that do not yet exist

## Protected continuity

The 13 Foundation Lock files, including `src/App.tsx`, `src/styles.css`, `src/main.tsx`, engine data and baseline screenshots, remain byte-identical.
