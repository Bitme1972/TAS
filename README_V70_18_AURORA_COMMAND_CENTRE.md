# TAS v70.18 AURORA Command Centre

This iteration preserves the full TAS v70.17 assurance and commercial parity surface while replacing the flat application shell with a modern security-engineering command centre.

## New experience layer

- Dynamic executive posture dashboard with telemetry debt, confidence, risk families and priority actions
- Four-stage assurance flight plan driven by live evidence state
- Evidence orchestration workspace with drag-and-drop multi-file intake
- Live evidence confidence, receipt hash, parser diagnostics and asset inventory
- Ctrl+K command palette for navigation, assessment, sample loading and output access
- Browser workspace autosave and restore using local storage
- Expandable or compact left navigation
- Responsive layout for desktop, laptop and smaller screens

## Validation

Run `npm run build`. The build executes TypeScript, Vite, Cloudflare preparation, every inherited v70 parity gate and the new AURORA innovation gate.
