TAS v70.20 Community - FIRST-TIME GITHUB SETUP
==========================================================

Folders:
C:\TAS\TAS_v70_19_COMMERCIAL_FOUNDATION   source package
C:\TAS\tas                                Git repository

For the first Git-only release, double-click RUN_ME_FIRST_TIME_GIT_SETUP.cmd.
This validates the protected AURORA core and all three independent editions before pushing.
It deliberately does not assume or change Cloudflare.

After the GitHub repository is connected to Cloudflare project tas, later releases may use
RUN_ME_DEPLOY_TO_GIT_AND_CLOUDFLARE.cmd or RUN_ME_DEPLOY_GIT_AND_CLOUDFLARE_DIRECT.cmd.

Cloudflare build command: npm run build
Cloudflare output directory: dist
Default Sprint 2 default public edition: TAS Professional
