TAS v70.18.1 Foundation Lock — CLEAN FIRST-TIME SETUP
============================================

Folder separation
-----------------
Keep the separate fashion project in its own existing folder.
C:\TAS is reserved exclusively for Telemetry Assurance Studio.

Use these two separate TAS folders:
C:\TAS\TAS_v70_18_1_FOUNDATION_LOCK   source package
C:\TAS\tas                 local clone of https://github.com/Bitme1972/tas.git

First release order
-------------------
1. Run RUN_ME_FIRST_TIME_GIT_SETUP.cmd
2. Confirm the v70.18.1 files appear in GitHub repository Bitme1972/tas
3. In Cloudflare create a new Pages project named tas from the existing Git repository Bitme1972/tas
4. Set Production branch to main
5. Set Build command to npm run build
6. Set Build output directory to dist
7. Leave Root directory blank
8. Save and Deploy
9. After the pages.dev site works, add auditpol.com under Custom domains

Future releases
---------------
Use RUN_ME_DEPLOY_TO_GIT_AND_CLOUDFLARE.cmd after the Cloudflare Git integration exists.
