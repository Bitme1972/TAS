TAS v70.20 Community - DEPLOYMENT GUIDE
===================================================

SAFE DEFAULT
------------
Run TEST_ALL_CAPABILITIES.cmd first. It is local-only and deploys nothing.

DIRECT GITHUB + CLOUDFLARE RELEASE
----------------------------------
1. Keep this package at:
   C:\TAS\TAS_v70_19_COMMERCIAL_FOUNDATION

2. Keep the Git clone separate at:
   C:\TAS\tas

3. Double-click:
   RUN_ME_DEPLOY_GIT_AND_CLOUDFLARE_DIRECT.cmd

The runner validates the package, locates or clones https://github.com/Bitme1972/tas.git,
protects uncommitted repository work, copies the complete TAS source including config and
all edition entry points, rebuilds the repository, verifies three edition manifests, commits,
pushes and deploys only the protected Professional dist folder to Cloudflare project tas.

DEFAULT PUBLIC OUTPUT
---------------------
Cloudflare receives dist only. During Sprint 2 this is the protected Professional/AURORA app.
Community and Consultant are compiled independently under dist-editions for validation and
local review but are not made public by the direct Cloudflare command.

NO-PUSH GUARANTEE
-----------------
If package validation, repository copy verification, any inherited test or any edition gate
fails before the Git commit stage, no commit or push occurs.
