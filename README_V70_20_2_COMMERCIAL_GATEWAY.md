# TAS v70.20.2 Commercial Gateway

This release keeps the protected AURORA commercial application intact while changing the public journey.

- `/` — Member Gateway and launch pricing
- `/demo/` — free interactive AURORA demonstration using fixed synthetic inputs
- `/studio/` — Professional and Consultant purchase/activation gateway
- `dist-editions/professional/` — protected Professional build for controlled fulfilment
- `dist-editions/consultant/` — protected Consultant build for controlled fulfilment

No real TAS Golden baseline is included in the Demo. No MSI was found in the supplied attachment, so the installer card remains honestly disabled until a signed installer is supplied.
