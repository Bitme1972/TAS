# TAS Interactive Demo security boundary

The free Demo is not the commercial product with buttons hidden.

It contains:

- the AURORA interaction pattern
- synthetic audit evidence
- synthetic demonstration controls
- synthetic Event ID context
- watermarked demonstration reports

It does not contain:

- the TAS commercial Golden baseline
- premium control identifiers or datasets
- customer evidence upload capability
- commercial usage rights
- a release-approved, hash-verified installer
- licence-signing keys, payment secrets or private entitlement data

The public `/studio/` route is a gateway only. The commercial application is supplied through an authenticated member download or private service after entitlement validation.
